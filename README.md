# مركز الكتاب للتحفيظ — PWA

تطبيق إدارة حلقات تحفيظ القرآن الكريم — يعمل كتطبيق كامل على الجوال.

## 📁 الملفات المطلوبة

```
index.html       ← التطبيق الرئيسي
manifest.json    ← معلومات التطبيق للجوال
sw.js            ← Service Worker (عمل أوفلاين)
icon-192.png     ← أيقونة صغيرة
icon-512.png     ← أيقونة كبيرة
```
---

## 📱 تثبيت التطبيق على الجوال

### أندرويد (Chrome):
1. افتح الرابط في Chrome
2. ستظهر رسالة **"إضافة إلى الشاشة الرئيسية"** تلقائياً
3. أو من قائمة Chrome ← **تثبيت التطبيق**
4. يُثبَّت كتطبيق كامل بدون شريط العنوان ✅

### iOS (Safari):
1. افتح الرابط في Safari
2. اضغط زر **المشاركة** (Share)
3. اختر **إضافة إلى الشاشة الرئيسية**
4. اضغط **إضافة** ✅

---

## ✅ مميزات هذا الإعداد

- يعمل **أوفلاين** بعد أول تحميل
- يُثبَّت كتطبيق كامل (بدون شريط المتصفح)
- البيانات محفوظة في **LocalStorage** على الجهاز
- مجاني 100% عبر GitHub Pages

---

## ⚠️ ملاحظة مهمة

البيانات تُحفظ على **جهاز المستخدم فقط**، وليس على الخادم.
لذلك كل مستخدم يرى بياناته الخاصة المحفوظة على جهازه.
